System requirements
-------------------
This program has only been tested on Linux but it should run on Windows and Mac OSX, too.

Software requirements
---------------------
You will need the listed software:
- Python (version 3)
- Python psutil
- psycopg (version 2)
- git
- PostgreSQL (server)
